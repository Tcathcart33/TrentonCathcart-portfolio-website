Site Control PLC Project

Complete design package for a PLC-based jail control system supporting a correctional facility environment. Personally responsible for project quoting, system design, and hands-on panel fabrication. Scope includes system architecture, PLC I/O design, power distribution (120VAC and 24VDC), intercom integration, and network infrastructure. Deliverables consist of detailed floor plans, one-line diagrams, enclosure layouts, rack configurations, field wiring schematics, and a full bill of materials. Panels were physically built and wired in-house, ensuring alignment between design intent and real-world implementation. The system enables centralized control and monitoring of cell devices, intercom stations, lighting, and security interfaces in a high-reliability, mission-critical application.
